# 💸 NetPay

A simple and intuitive **person-to-person expense tracking app** that helps you manage shared expenses and track who owes whom — all in one place.

---

## 🚀 Features

* 👤 User authentication (Firebase Auth)
* 📊 Real-time expense tracking
* 🤝 Add friends and manage balances
* ➗ Split expenses (Manual / Equal)
* 💰 Clear “They owe me” / “I owe them” system
* 📜 Transaction history (ledger view)
* 🌙 Clean dark UI

## 🧱 Tech Stack

* **Frontend:** Android (Kotlin)
* **Backend:** Firebase

  * Cloud Firestore
  * Firebase Authentication
* **Architecture:** MVVM (if used)

---

## 🔥 How It Works

1. User signs in
2. Add friends
3. Create a new expense
4. Split amount (manual / equally)
5. App updates balances automatically
6. View ledger to track all transactions
