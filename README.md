# NetPay

NetPay is a simple Android app to track shared expenses and manage balances. It helps you keep track of who owes whom without handling actual payments.

##  Features
- Add and track expenses
- View balances (who owes whom)  
- User authentication with Firebase  
- Simple and clean interface  

##  Tech Stack
- Kotlin  
- Android SDK  
- Firebase Authentication  

##  How it Works
1. Sign up / log in  
2. Add expenses and log a transaction  
3. View updated balances instantly
